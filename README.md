## Week 1 — Contents

### Topic 1: OLAP vs OLTP

Research and comparative analysis of the two fundamental paradigms for data processing in information systems.

| Feature | OLTP | OLAP |
|---|---|---|
| Purpose | Operational transactions | Complex analysis & queries |
| Data type | Current / operational | Historical / multidimensional |
| Query style | Simple, fast | Complex, aggregations |
| Typical users | Operations staff | Analysts / executives |
| Example | Sales system | Data Warehouse |
